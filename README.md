# 📚 Literalura

Aplicación desarrollada en **Java con Spring Boot** que permite consultar libros desde la API pública Gutendex y almacenarlos en una base de datos PostgreSQL.

Este proyecto forma parte del programa de formación **Oracle Next Education** y **Alura Latam**.

---

## 🚀 Tecnologías utilizadas

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- PostgreSQL
- API Gutendex
- Maven

---

## ⚙️ Funcionalidades

La aplicación funciona desde la consola y permite:

1. Buscar libro por título
2. Listar libros registrados
3. Listar autores registrados
4. Listar autores vivos por año
5. Listar libros por idioma

Los libros buscados se obtienen desde la API Gutendex y se almacenan en una base de datos PostgreSQL.

---

## 🌐 API utilizada

Este proyecto consume datos desde la API pública:

https://gutendex.com/

La API proporciona información sobre miles de libros del proyecto Gutenberg.

---

## 🗄️ Base de datos

Se utiliza **PostgreSQL** para almacenar:

- Libros
- Autores
- Relación entre libros y autores
